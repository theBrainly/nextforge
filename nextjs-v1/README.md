# NextJS V1 Starter

Production-ready Next.js 14+ boilerplate with App Router and TypeScript.

## Features
- App Router with strict TypeScript
- Tailwind CSS + responsive UI
- Auth pages (login/register)
- Protected dashboard layout
- API health check route
- Axios client with interceptors
- Zustand state management
- React Hook Form + Zod validation
- Dark/light theme toggle
- Global toast notifications
- Loading and error boundaries
- ESLint + Prettier + Husky + lint-staged

## Setup
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Scripts
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run format`

## Folder Architecture
Follows the required enterprise-ready structure under `src/` with app routes, feature modules, shared components, and centralized configs.
